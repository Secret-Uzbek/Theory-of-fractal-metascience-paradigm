> Core theory-facing repository for the Fractal Metascience Paradigm: the
> paper-level and framework-level public entry into the theoretical branch of
> the wider Terra ecosystem.

## Layer role

This repository is the explicit theory layer of the Terra / FMP GitHub weave.

It should serve as:

- a public theoretical entry point;
- a theory-facing bridge between the central hub and deeper research branches;
- a bridge from the central hub toward the deeper theoretical corpus.

It should not remain a broken auto-generated shell with mojibake and weak
service files.

It should also not collapse publication packaging, archive residue, and machine
maps into the same first human-facing surface.

## Reading path

If you are a human reader, start with:

1. this `README.md`
2. `CITATION.cff`
3. `Theory/` and root theory documents
4. donor rules in `terra-legal`
5. publication-facing companion repositories if you need release or DOI context

## Ecosystem position

This repository belongs to the wider FMP/Terra contour:

- `FMP-CENTRAL-REPO` — central public hub
- `terra-legal` — donor legal/governance layer
- `Nullo-PLT-UCOMM-FMP-Academic-Research` — academic bridge repository
- `FMP-monograph` — long-form monograph layer
- `FMP-Full-Publication-Package` — publication package layer

## Layer boundaries

Keep here:

- theory documents;
- conceptual and analytical framework material;
- theory-facing public pages and metadata;
- theory-linked release and citation surfaces.

Do not overload this repository with:

- raw archive dumps;
- branch-specific practical implementation files;
- donor governance files rewritten from scratch;
- machine traces pretending to be the first entry point.
