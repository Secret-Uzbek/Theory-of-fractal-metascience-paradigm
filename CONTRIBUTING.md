# Contributing

This repository is for theory-facing cleanup, integrity, and publication-grade
readability.

Good contribution types:

- README and documentation repair;
- mojibake removal;
- citation and metadata repair;
- theory-facing structure cleanup;
- explicit distinction between public human files and machine or donor layers.

Avoid:

- noisy automation output in public root files;
- fake “finished” placeholders;
- commits that make the theory surface less readable than before.

Before contributing, read:

- `README.md`
- `CITATION.cff`
- `GOVERNANCE.md`
- `terra-legal` as the donor legal/governance layer
